# Mercury Vibecoding Project

## 项目简介

本项目为课程小组作业，目标是在 GitHub 开源项目 Mercury 的基础上，通过 Vibecoding 的方式复刻其核心功能。

参考项目：https://github.com/neolee/mercury

项目定位：一个本地优先、跨平台、支持 AI 摘要 / 翻译 / 标签功能的 RSS 阅读器。

## 作业要求功能

- Feed / OPML 解析
- Feed 同步
- 内容清洗：Cleaned HTML / Cleaned Markdown
- 文章内容展示
- Summary Agent
- Translation Agent
- LLM Providers
- 多语言支持
- 日志上报和调试工具
- 大语言模型用量统计
- 笔记和文摘导出
- 标签系统：文章标签、按标签筛选、Tag Agent、标签管理

---

## 技术约束

- 本地优先
- 无需注册登录或订阅
- 不主动采集用户数据
- 支持 Windows / Linux / macOS
- 支持标准 API 的大语言模型服务，包括本地模型

## 项目计划

### 第一阶段：需求分析与项目初始化

- 阅读 Mercury 项目
- 整理功能需求
- 确定技术栈
- 初始化 GitHub 仓库

### 第二阶段：基础功能开发

- 实现 Feed 添加与解析
- 实现文章列表与正文展示
- 实现本地数据库存储
- 实现 OPML 导入导出

### 第三阶段：AI 功能开发

- 实现 LLM Provider 配置
- 实现文章摘要
- 实现文章翻译
- 实现 AI 标签推荐
- 统计模型调用用量

### 第四阶段：完善与展示

- 实现笔记和文摘导出
- 完善标签管理
- 补充日志与调试工具
- 编写项目文档和展示材料
